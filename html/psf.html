PSF¶

class piff.PSF(model, interp)¶

A class that encapsulates the full interpolated PSF model.

The usual way to create a PSF is through one of the two factory functions:

>>> psf = piff.PSF.build(images=images, pos=pos, model=model, interp=interp, ...)
>>> psf = piff.PSF.read(file_name=file_name, ...)

The first is used to build a PSF model from the data. The second is used to read in a PSF model from disk.

However, it is also possible to construct a PSF from a model instance and an interp instance. Any existing parameters in the model instance are ignored. The model just determines how interpolated parameters are to be interpreted.

classmethod build(stars, model, interp, logger=None)¶

The main driver function to build a PSF model from data.

Parameters:
  • stars – A list of StarData instances.
  • model – A Model instance that defines how to model the individual PSFs at the location of each star.
  • interp – An Interp instance that defines how to do the interpolation of the data vectors (produced by model for each star).
  • logger – A logger object for logging debug info. [default: None]
Returns:

a PSF instance
